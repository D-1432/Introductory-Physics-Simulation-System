简介
本项目是一个用于模拟粒子在重力场、磁场以及两者耦合场中运动的二维仿真系统。用户可以通过交互式菜单创建和管理多个粒子（物体），设置其初始位置、速度、质量和电荷量，选择物理场并运行仿真。仿真结果将输出为数据文件，并可调用 Gnuplot 自动绘制运动轨迹。

主要功能
物体管理：支持增加、删除、修改、查找和显示粒子信息。每个粒子包含编号、质量、坐标、速度、电荷量等属性。

物理场仿真：

重力场：仅受恒定重力作用（G = -9.8，可配置），使用匀加速运动公式更新位置和速度。

磁场：仅受垂直纸面向外的均匀磁场作用，粒子做匀速圆周运动，采用解析解更新位置。

耦合场：同时受重力和磁场作用，采用叠加方式更新（目前为简单叠加，注意物理合理性）。

数据输出：仿真结果按每个物体保存为独立的文本文件（如 G_run_0.txt、B_run_1.txt、run_2.txt），格式为 t x y vx vy。

自动绘图：利用 Gnuplot 绘制轨迹图，可选择单个物体或所有物体在同一图中显示。

持久化存储：物体信息保存至 m_Info.txt，下次启动时自动加载。

依赖环境
C++ 编译器（支持 C++11 及以上标准）

Gnuplot（用于绘图，需将 gnuplot 添加到系统 PATH）

文件结构
文件名	作用
仿真系统.cpp	主程序入口，显示菜单并调用 Objects 类功能
Objects.h / Objects.cpp	物体管理类，维护物体数组、文件读写、增删改查等
Body.h / Body.cpp	物体基类，包含基本属性（质量、坐标、速度、编号、名称、电荷）
Particle.h / Particle.cpp	粒子类，继承自 Body，目前仅用于区分类型
Field.h / Field.cpp	物理场基类，继承自 Objects，提供仿真数据绘图功能
Gravity.h / Gravity.cpp	重力场类，实现重力场仿真逻辑
Magnetic.h / Magnetic.cpp	磁场类，实现磁场仿真逻辑
Multi-field coupling.h / Multi-field coupling.cpp	耦合场类，实现重力与磁场同时作用的仿真
参数控制.h	定义全局常量：重力加速度 G、时间步长 DT、磁场强度 BOT（未使用）、文件名宏
标头.h	包含常用头文件（仅为简化主程序包含）
m_Info.txt	默认的物体信息存储文件（自动生成）
*_run_*.txt	仿真结果数据文件（自动生成）
编译与运行
编译
使用支持 C++11 的编译器编译所有 .cpp 文件，例如：

g++ -std=c++11 仿真系统.cpp Objects.cpp Body.cpp Particle.cpp Field.cpp Gravity.cpp Magnetic.cpp Multi-field\ coupling.cpp -o simulation
运行
在终端中执行生成的可执行文件：

./simulation
按菜单提示操作即可。

使用说明
首次运行：由于 m_Info.txt 不存在，系统会提示文件不存在，您需要先添加物体（选择菜单 1）。

添加物体：输入数量、编号、质量、坐标、速度、电荷量，并选择类型（目前仅支持“粒子”）。

运行仿真：

选择菜单 6 → 输入仿真类型（1=重力场，2=磁场，3=耦合场）。

输入仿真步数（每步时间间隔由 DT 决定）。

仿真结束后可选择是否绘图（1=是，2=否）。

绘图：绘图将弹出 Gnuplot 窗口，显示物体轨迹。按 Enter 关闭窗口返回菜单。

数据文件：所有仿真数据保存在以 场名_run_编号.txt 命名的文件中，可用文本编辑器查看。

注意事项
磁场仿真中，粒子电荷量 q 不能为零，否则会导致除零错误。

耦合场仿真目前采用简单叠加，可能不符合严格的物理方程，仅供演示。

绘图功能依赖 Gnuplot，若未安装或未配置 PATH，将提示失败。

文件 m_Info.txt 存储格式为：编号 类型名称 质量 X Y Vx Vy 电荷量，请勿手动修改格式以免读取错误。
